# BMI Calculator

A simple and modern BMI Calculator in Arabic.

<img width="790" height="529" alt="image" src="https://github.com/user-attachments/assets/f6c9b591-8c71-438f-b211-55154174611c" />


## Features
- Enter weight (kg) and height (cm)
- Calculates BMI and shows a description: Underweight, Healthy, or Overweight
- Colors change based on the result
- Responsive design for mobile and desktop

## How to Use
1. Open `index.html` in your browser
2. Enter your weight and height
3. Click "احسب BMI"
4. See your BMI number and result description
